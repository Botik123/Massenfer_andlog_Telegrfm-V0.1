Привествую, что бы все работала успешно и без багов, в корне должно быть 2 папки (Client и server), bat-ник запускает сервера


Messenger/
│
├── 📁 server/                          # Серверная часть
│   ├── 📄 index.js                     # Главный сервер (Express + WebSocket)
│   ├── 📄 db.js                        # База данных SQLite
│   ├── 📄 package.json                 # Зависимости сервера
│   ├── 📄 package-lock.json            # Фиксация версий (создаётся автоматически)
│   ├── 📁 node_modules/                # Библиотеки сервера (автоматически)
│   ├── 📁 uploads/                     # Папка с загруженными файлами
│   │   ├── 🖼️ *.jpg, *.png             # Изображения
│   │   ├── 🎬 *.mp4                    # Видео
│   │   ├── 🎵 *.mp3                    # Аудио
│   │   ├── 📄 *.pdf, *.doc             # Документы
│   │   └── 🖼️ thumb_*.jpg              # Миниатюры изображений
│   └── 🗄️ messenger.db                 # Файл базы данных (создаётся автоматически)
├── 📁 client/                          # Клиентская часть (React)
│   ├── 📁 public/
│   │   └── 📄 index.html               # Главная HTML страница
│   ├── 📁 src/
│   │   ├── 📄 index.js                 # Точка входа React
│   │   ├── 📄 App.js                   # Главный компонент приложения
│   │   ├── 📄 App.css                  # Стили приложения
    ├── 📄 package.json                 # Зависимости клиента
│   ├── 📄 package-lock.json            # Фиксация версий
│   └── 📁 node_modules/                # Библиотеки клиента (автоматически)
│
├── 📄 start.bat                        # Скрипт быстрого запуска (Windows)
